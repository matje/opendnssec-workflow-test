This project's goal is the implementation of CGA-TSIG draft. This project is the result of a cooperative effort between NL NET LAB and Hasso Plattner Institute. The purpose of the implementation, which is the outcome of this project, is to provide security for  the Domain Name System (DNS) during authentication processes. It can be applied for different scenarios based on what the desired end result is.
